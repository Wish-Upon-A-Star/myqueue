# 작업 큐 탐색기

이 프로젝트는 브라우저에서 동작하는 **작업/큐 탐색기 단일 HTML 페이지**입니다.  
별도 설치 없이 `task_explorer.html` 하나만으로 작업 트리 구조를 만들고 관리할 수 있습니다.

## 주요 기능

- 트리 기반 작업 관리
  - 작업 추가 / 수정 / 삭제
  - 하위 작업(서브태스크) 추가
  - 부모-자식 구조로 이동
- 뷰 전환
  - 전체
  - 오늘 할일
  - 중요
  - 날짜별 보기
  - 폴더별 보기
  - 목록별 보기
- 작업 조작
  - 우선순위 지정 및 정렬
  - 작업 완료 처리 (우클릭)
  - 메모 작성
  - 중요 표시(별)
- 드래그 앤 드롭
  - 작업 재정렬
  - 폴더/목록으로 이동
  - 폴더끼리/목록끼리 정렬
- 보존 기능
  - 브라우저 로컬스토리지 자동 저장
  - 상태 파일 저장/불러오기(JSON)
  - 현재 구조를 텍스트 트리로 내보내기(`.txt`)

## 실행 방법

### 1) 파일 실행(가장 간단)
1. `task_explorer.html` 더블클릭
2. 브라우저에서 페이지가 열리면 바로 사용 가능

### 2) 로컬 서버로 실행(권장)
브라우저 보안 정책 때문에 일부 기능은 서버 환경에서 더 안정적입니다.

- Python이 설치된 경우
  - PowerShell에서 아래 명령 실행
  - `python -m http.server 8000`
  - 브라우저에서 `http://127.0.0.1:8000/task_explorer.html` 접속

## 파일 구성

- `task_explorer.html`
  - 전체 UI와 기능이 들어 있는 단일 페이지

## 데이터 저장 위치

- 브라우저 localStorage: 브라우저를 다시 열어도 작업 데이터가 유지됩니다.
- 파일 백업: `폴더에 저장` 버튼으로 JSON 파일로 내보내기
- 복원: `폴더에서 불러오기` 또는 파일 선택 기능

## 알려진 사용 포인트

- 폴더/목록/날짜 패널은 버튼을 눌러 열고 닫는 방식입니다.
- 트리 텍스트 내보내기는 현재 선택된 루트(현재 경로) 기준으로 내려갑니다.
- 하위가 없는 경우에도 현재 루트 기준 구조가 안전하게 처리됩니다.

## 권장 브라우저

- 최신 Chrome / Edge / Firefox
